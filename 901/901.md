Do enunciado, temos:

Plano K:\
Para x impulsos, temos:

$K(x) =$ $29 ,90 \\, se\\, 0 \leq x \leq 200$\

Plano Z:\
Para x impulsos, temos:

$Z(x) =$ $49,90\\, se\\, 0 \leq x \leq 300$\
            $0,10x + 49,90 \\, se\\, x > 300$
